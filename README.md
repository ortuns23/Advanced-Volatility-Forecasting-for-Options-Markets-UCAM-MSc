# Advanced-Volatility-Forecasting-for-Options-Markets-UCAM-MSc
Proyecto Final de Máster Excepcional (100/100): Investigación cuantitativa avanzada en previsión de volatilidad para mercados de opciones. Implementa y compara modelos de la familia GARCH y ML (LSTM, XGBoost) con validación estadística rigurosa. Aplicación directa a gestión de riesgos y estrategias de trading algorítmico.
